# Task Tracker Client

https://comp-2068-js-frameworks-o5qp-kh9ddwp5s-umaid-devs-projects.vercel.app/

## Description
This is a React client application built using Next.js.  
It allows users to create, view, update, and delete tasks.

## Features
- View all tasks
- View task details
- Create new tasks
- Edit tasks
- Delete tasks
- Responsive navigation bar

## Pages

- Home
- List Tasks
- Add Task
- Edit Task
- Task Details

## Tech Stack
- React (Next.js)
- TypeScript
- Fetch API
